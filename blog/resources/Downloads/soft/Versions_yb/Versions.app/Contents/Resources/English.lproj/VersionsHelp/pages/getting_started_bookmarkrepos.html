Bookmarking a repository

Follow these steps to add a bookmark to an existing repository in Versions.

  • Find your repository URL.
  • Click “New Repository Bookmark” in the welcome screen.
  • Type in a name for the repository bookmark.
  • Paste or type in the repository URL.
  • If needed, enter a username and password.
  • Click the “Create” button and voilá.

Versions verifies that the repository can be accessed and adds it to your bookmarks list. For details on setting up repository bookmarks with custom port numbers and using ssh key-pairs to authenticate, please refer to the Advanced topics section.

To bookmark a local repository, click the small folder button next to the “Location” field, and choose the folder from the dialog that opens.
