Work with your working copy

The Browse view shows the contents of your working copy. The leftmost column in the table shows an icon representing the local status for each file and folder. Hover over the icon or resize the column to reveal a text label.

The leftmost column shows local status, the one next to it shows remote status.

Every item in a working copy has a so-called Subversion status. Some files are modified, others may be added, ignored or unversioned. See this page in the online SVN Book for a complete list of possible statuses. You can filter the working copy browser to only show items with changes by selecting “Changed” in the switch at the top-right corner.

To keep everything snappy, Versions only shows the status of local files in the browse view by default. Select “Show Updates from Repository” from the “View” menu to make Versions display the status of files in the repository.
