Local repositories

Versions also lets you create a local repository on your own computer. Local repositories cannot easily be shared with others, but they work just fine if you want to use Subversion to manage a project you’re working on by yourself. To create a local repository follow these steps:

  • Choose “Create Local Repository...” under the “Bookmarks” menu.
  • Navigate to where you want to create your repository.
  • Enter a name for the repository and click “Create”.

Your local repository is automatically added to the bottom of the bookmarks list.
