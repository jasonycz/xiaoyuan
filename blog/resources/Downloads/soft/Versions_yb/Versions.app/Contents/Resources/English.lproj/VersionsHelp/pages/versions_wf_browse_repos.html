Browse repositories

Another use for the browser is to browse through a repository directly. When you select a repository in the bookmarks list, you’ll notice that the appearance of the browser changes slightly. The status columns that are visible when you browse a working copy are not shown while browsing a repository. Double-clicking a file in the repository browser will fetch a copy from the server and open it, while double-clicking a folder will disclose or collapse it.

The Revision control in the top-right corner lets you choose any earlier revision of your repository. After selecting a folder in the browser, click “Checkout” to create a new working copy from the revision you are browsing. The new working copy is automatically added to the bookmarks list. Use “Export...” under the “File” menu to create an unversioned local copy of a file or folder.
