Checkout a working copy

Follow these steps to checkout a fresh working copy from a repository you have bookmarked in Versions.

  • Select your repository in the bookmarks list.
  • Choose the “Browse” tab in the main window.
  • Select the repository's “trunk” or any other folder you want to create a working copy of.
  • Click the “Checkout” button.
  • Navigate to where you want to save your working copy.
  • Name your working copy and click “Checkout” in the dialog.

Versions will checkout a new working copy at the location you specified. Depending on how large the reopsitory is, this can take quite a while. When Versions is done checking out the working copy, it is automatically added to the bookmarks list under the appropriate repository.
