Importing files into an existing repository

Follow these steps to import files on your Mac into a repository you have already bookmarked in Versions.

  • Select your repository in the bookmarks list, and browse to the folder into which you want your files to be imported.
  • Select the "File > Import..." menu item. Alternatively, you can right-click the selected folder and select "Import..." from the contextual menu, or click the "Import" button in the toolbar.
  • Choose the folder to import by clicking on the small folder button.
  • Choose the destination folder from the repository outline shown below. By default Versions will import the contents of the source folder into the destination folder. You can optionally specify a new folder with a different name for the imported folder in the "New Folder Name:" field.
  • Click the "Import" button.
  • You will be prompted for a log message where you can optionally enter text describing why you are importing the files before clicking "Commit".
