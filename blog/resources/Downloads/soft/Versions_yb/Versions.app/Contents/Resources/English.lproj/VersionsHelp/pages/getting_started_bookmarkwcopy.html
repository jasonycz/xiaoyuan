Adding an existing working copy

Follow these steps if you have an existing working copy that you want to import into Versions.

  • Click “New Working Copy Bookmark” in the welcome screen.
  • Click the little folder icon next to “Select Folder...”.
  • Locate the working copy and click “Choose”.

The imported working copy is automatically added under the correct repository in the bookmarks list. If the repository isn’t bookmarked yet, it will be added automatically at the bottom of the bookmarks list, with the working copy under it.

Alternatively, you can just drag a working copy folder from the Finder into the bookmarks list in Versions. Versions will create the correct bookmark, and will add a repository bookmark if necessary.
