Changing properties

In most programming projects, you’ll have some files you do not want to keep under version control, like your build folder. The simplest way to ignore this folder is to control+click it in the browser and select “Ignore "build"” from the contextual menu. To review and set other properties for files and folders, click the “Inspector” button in the toolbar. If you select the folder that contains the build folder, you’ll notice that "build" is listed in the “Ignore” list.
