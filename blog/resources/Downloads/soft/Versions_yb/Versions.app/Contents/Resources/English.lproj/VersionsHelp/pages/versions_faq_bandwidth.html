Bandwidth

Why does Versions use bandwidth when idle?

Versions doesn't only use bandwidth when it's committing or updating changes to and from your repository. Versions also uses bandwidth for updating the bookmarks badges in the bookmarks list. If you have a very large project, these checks can result in Versions having to download a lot of data to accurately figure out the amount of changes.

You can disable "Show badges with working copy changes" in preferences to help improve performance.
