Browse

Versions’ “Browse” interface lets you interact with working copies and directly with repositories. The behavior and features of the browser vary slightly for both. Most sections below are applicable to using with a working copy, while the last section covers using the browser with a repository.

  • Work with your working copy
    Keep track of local and remote changes in your working copy.
  • Committing changes
    Adding files and committing changes.
  • Updating and reverting
    Update all or part of your working copy, or undo local changes.
  • Changing properties
    Ignore specific files and folders, inspect properties.
  • Browse repositories
    Browse any revision of a repository directly, create working copies.
  • Versions badges and icons
    Learn about the different badges and icons in Versions.
