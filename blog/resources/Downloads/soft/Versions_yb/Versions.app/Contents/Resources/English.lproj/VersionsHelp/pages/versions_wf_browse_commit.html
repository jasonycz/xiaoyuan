Committing changes

To add a new file in your working copy to your repository, select the file in the browser and click the “Add” button in the toolbar to schedule the new file for addition. You can select multiple unversioned files to add them all at once. Next, click the “Commit” button to send your changes to the repository.

The buttons in Versions’ toolbar are enabled or disabled depending on what is selected.
