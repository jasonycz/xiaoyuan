Grammaires

Une spécification de format de fichier peut inclure une définition de grammaire utilisée pour le surlignage syntaxique et pour aider à définir quelles différences sont importantes.

Par exemple, notre format standard pour le code source C/C++ inclut une grammaire qui reconnait les mots-clés et les chaînes de caractères littérales.  En indiquant les différences dans les commentaires comme non importantes, vous pouvez mieux vous concentrer sur les différences importantes.

Une définition de grammaire dans Beyond Compare est assez simple.  Il ne supporte pas les définitions imbriquées ou propre au contexte, ainsi il ne prend pas très bien en compte les langages avec des balises tels que HTML et PHP.  D'un autre côté, il s'agit d'une grande amélioration des fonctionnalités de Beyond Compare 2.

Pour modifier une grammaire, sélectionnez Outils > Formats de fichier et prenez le format approprié de fichier.  Allez à l'onglet Grammaire et modifiez les éléments de grammaire.
