Commandes de Comparaison Hexadécimale

Ce qui suit est une description des commandes de Comparaison Hexadécimale qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de Comparaison Hexadécimale.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Comparaison Hexadécimale vide.

Permuter les côtés

Échange les fichiers de gauche et de droite.

Recharger les fichiers

Recharge les fichiers affichés en demandant de les enregistrer avant si cela est nécessaire.

Recomparer les fichiers

Recompare les fichiers affichés, en préservant les modifications, mais pas les possibilités de retour arrière.

Rapport de Comparaison Hexadécimale

Produit le Rapport de Comparaison Hexadécimale.

Informations de Comparaison Hexadécimale

Affiche les statistiques de comparaison.

Comparer des fichiers en utilisant

Ouvre les fichiers affichés dans une nouvelle session en utilisant une classe différente de vue de fichiers.

Comparer les dossiers parents

Ouvre une nouvelle session de Comparaison de Dossiers basée sur les dossiers parents des fichiers actuellement chargés.

Le menu Fichier

Ouvrir un fichier

Ouvre un fichier existant dans le panneau sélectionné.

Ouvrir un fichier du système de fichier *

Ouvre un fichier local existant dans le panneau sélectionné.

Ouvrir un fichier en utilisant un profil *

Ouvre un fichier existant dans le panneau sélectionné en utilisant un profil.

Ouvrir le presse-papiers

Charge le contenu du presse-papiers dans le panneau sélectionné.

Ouvrir avec

Ouvre le fichier du panneau sélectionné avec une des applications externes Ouvrir avec.

Ouvrir avec >

  Application associée

Ouvre le fichier du panneau sélectionné avec son application associée.

Ouvrir avec >

  Éditeur de texte

Ouvre le fichier du panneau sélectionné avec l'éditeur de texte intégré.

Enregistrer le fichier

Sauvegarde le fichier modifié du panneau sélectionné.

Enregistrer le fichier sous

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom.

Enregistrer le fichier dans le système de fichier sous *

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom sur le système de fichier.

Enregistrer le fichier en utilisant le profil sous *

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom en utilisant un profil.

Explorateur

Affiche le menu contextuel du gestionnaire de fichier pour le fichier du panneau sélectionné.

Le menu Edition

Défaire

Défait l'action précédente.

Refaire

Refait une action précédemment défaite.

Copier vers la droite / Copier vers la gauche *

Copie les octets sélectionnés vers le côté indiqué.

Copier vers l'autre côté

Copie les octets sélectionnés vers le côté opposé.  Le titre et l'icône apparaissent comme Copier vers la droite ou Copier vers la gauche en fonction du côté actif.

Couper / Copier / Coller / Supprimer

Commandes d'édition standard.

Sélectionner tout

Sélectionne tous les octets visibles dans le panneau actif.

Le menu Recherche

Octets différents suivants

Se positionne sur les octets différents suivants.

Octets différents précédents

Se positionne sur les octets différents précédents.

Fichiers différents suivants

Ouvre le couple suivant de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Fichiers différents précédents

Ouvre le couple précédent de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Copier le fichier vers la droite et ouvrir le différent suivant

Copie le fichier entier du côté gauche vers la droite et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Copier le fichier vers la gauche et ouvrir le différent suivant

Copie le fichier entier du côté droit vers la gauche et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Rechercher

Recherche un texte correspondant.

Remplacer

Remplace le texte correspondant.

Rechercher le suivant

Recherche l'occurrence suivante de la chaîne de caractères de recherche.

Rechercher le précédent

Recherche l'occurrence précédente de la chaîne de caractères de recherche.

Aller en

Se positionne sur un octet spécifié.

Le menu Affichage

Afficher tout

Affiche tous les octets, avec ou sans différences.

Afficher les différences

Affiche uniquement les octets avec des différences.

Afficher les identiques

Affiche uniquement les octets sans différence.

Adresse de l'octet actif

Affiche l'adresse de l'octet actif.

Valeurs Little endian

Liste les valeurs en traitant le premier octet comme étant le moins significatif.

Valeurs Big endian

Liste les valeurs en traitant le premier octet comme étant le plus significatif.

Adresses d'octet

Affiche ou cache les adresses d'octet.

Police d'affichage *

Sélectionne la police à utiliser pour afficher les données de fichier.

Disposition

Configure les options de disposition.

Disposition côte à côte

Arrange les panneaux à gauche et à droite l'un de l'autre.

Disposition dessus dessous

Arrange les panneaux au-dessus en au-dessous l'un de l'autre.

Miniatures

Affiche ou cache le résumé miniature des changements à gauche de la vue.

Informations de fichier

Affiche ou cache les panneaux d'information de fichier au-dessus des panneaux.

Barre d'outils

Affiche ou cache la barre d'outils.

* Caché par défaut.
