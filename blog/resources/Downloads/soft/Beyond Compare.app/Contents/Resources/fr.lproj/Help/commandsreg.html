Commandes de Comparaison de Registres

Ce qui suit est une description des commandes de Comparaison de Registres qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de la Comparaison de Registre.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Comparaison de Registres vide.

Permuter les côtés

Échange les registres de gauche et de droite.

Niveau supérieur >

  Côté gauche / Côté droit / Les deux côtés

Change l'une ou l'autre, ou les deux clés de base en leur clé parente.

Recharger

Recharge les registres affichés.

Recomparer

Recompare les registres affichés sans les recharger.

Rapport de Comparaison de Registres

Produit le Rapport de Comparaison de Registres.

Informations de Comparaison de Registres

Affiche les statistiques de comparaison.

Comparer des fichiers en utilisant

Ouvre les fichiers affichés dans une nouvelle session en utilisant une classe différente de vue de fichiers.

Comparer les dossiers parents

Ouvre une nouvelle session de Comparaison de Dossiers basée sur les dossiers parents des fichiers actuellement chargés.

Le menu Fichier

Ouvrir

Ouvre un registre dans le panneau sélectionné.

Ouvrir un registre *

Ouvre un registre actif dans le panneau sélectionné.

Ouvrir un fichier registre du système de fichier *

Ouvre un fichier de registre local existant dans le panneau sélectionné.

Ouvrir un fichier de registre en utilisant un profil *

Ouvre un fichier de registre existant dans le panneau sélectionné en utilisant un profil.

Ouvrir avec

Ouvre le fichier du panneau sélectionné avec une des applications externes Ouvrir avec.

Ouvrir avec >

  Application associée

Ouvre le fichier du panneau sélectionné avec son application associée.

Ouvrir avec >

  Éditeur de texte

Ouvre le fichier du panneau sélectionné avec l'éditeur de texte intégré.

Explorateur

Affiche le menu contextuel du gestionnaire de fichier pour le fichier du panneau sélectionné.

Le menu Edition

Définir comme clés de base

Définit les clés sélectionnées comme clés de base.

Définir les deux comme clés de base

Définit la clé sélectionnée et la clé de l'autre côté comme clés de base.

Définir comme clé de base pour l'autre côté

Définit la clé sélectionnée comme clé de base pour l'autre côté.

Défaire

Défait l'action précédente.

Refaire

Refait une action précédemment défaite.

Copier vers la droite / Copier vers la gauche *

Copie les éléments sélectionnés vers le côté indiqué.

Copier vers l'autre côté

Copie les lignes sélectionnées vers le côté opposé.  Le titre et l'icône apparaissent comme Copier vers la droite ou Copier vers la gauche en fonction du côté actif.

Copier

Copie les lignes sélectionnées dans le presse-papiers.

Supprimer

Supprime les éléments sélectionnés.

Renommer

Renomme l'élément actif.

Nouvelle clé

Ajoute une nouvelle clé.

Nouvelle valeur

Ajoute une nouvelle valeur.

Modifier

Modifie la valeur actuelle.

Copier le nom de la clé

Copie le nom de la clé active dans le presse-papiers.

Exporter

Enregistre la clé active comme un fichier registre.

Exporter tout

Enregistre toutes les clés comme un fichier registre.

Sélectionner tout

Sélectionne toutes les lignes visibles dans le panneau actif.

Le menu Recherche

Élément différent suivant

Se positionne sur l'élément différent suivant.

Élément différent précédent

Se positionne sur l'élément différent précédent.

Fichiers différents suivants

Ouvre le couple suivant de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Fichiers différents précédents

Ouvre le couple précédent de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Copier le fichier vers la droite et ouvrir le différent suivant

Copie le fichier entier du côté gauche vers la droite et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Copier le fichier vers la gauche et ouvrir le différent suivant

Copie le fichier entier du côté droit vers la gauche et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Rechercher

Recherche un texte correspondant.

Rechercher le suivant

Recherche l'occurrence suivante de la chaîne de caractères de recherche.

Rechercher le précédent

Recherche l'occurrence précédente de la chaîne de caractères de recherche.

Le menu Affichage

Afficher tout

Affiche tous les éléments, avec ou sans différence.

Afficher les différences

Affiche uniquement les éléments avec des différences.

Afficher les identiques

Affiche uniquement les éléments sans différence.

Afficher aucun *

Cache tous les éléments.

Développer tout

Ouvre tous les éléments.

Réduire tout

Ferme tous les éléments.

Espace blanc visible

Affiche ou cache les espaces, tabulations.

Police d'affichage *

Sélectionne la police à utiliser pour afficher les données de fichier.

Disposition côte à côte

Arrange les panneaux à gauche et à droite l'un de l'autre.

Disposition dessus dessous

Arrange les panneaux au-dessus en au-dessous l'un de l'autre.

Détails en texte

Affiche ou cache le panneau des détails en texte en bas de la vue.

Détails en hexadécimal

Affiche ou cache le panneau des détails en hexadécimal en bas de la vue.

Informations de fichier

Affiche ou cache les panneaux d'information de fichier au-dessus des panneaux.

Barre d'outils

Affiche ou cache la barre d'outils.

* Caché par défaut.
