Comment déverrouiller Beyond Compare

Aide > Entrez la clé

Beyond Compare est distribué en mode évaluation, cela vous permet de voir s'il répond à vos besoins.  Sauf si vous achetez une licence, le programme cessera de fonctionner après 30 jours d'utilisation.

Lorsque vous achetez une licence de Beyond Compare, nous vous envoyons une clé de déblocage par courrier électronique.  Elle ressemble à quelque chose comme ceci :

  --- BEGIN LICENSE KEY ---

  CUgJWTb0eQVybpezCYu9CSqDLCtvx6g63hp-hrKHXiNL144KP4FTWHl9I

  q4+F5RWwcpzSyX98tlDzSsJm5xvnjuvunNrAG16sjP8Dp4as1gP3M1Pkj

  dmDVZXXLlyHoIrMd-Ptl0wpWPU654xu8kc7iYPl5LhOfXJG64V12vJnOm

  --- END LICENSE KEY -----

 

1.À partir de votre programme de courriel, sélectionnez la clé et copiez-la vers le presse-papiers.  Ce n'est pas grave si vous incluez quelques morceaux de message avant et après la clé, mais soyez certain d'inclure la clé entière.
2.Ensuite, exécutez Beyond Compare et sélectionnez l'option Entrer la clé.  (Cliquez sur le bouton Entrer la clé de n'importe quel dialogue "Évaluation", ou sélectionnez Aide > Entrer la clé.)
3.Dans le dialogue "Enregistrer Beyond Compare", votre clé devrait apparaître automatiquement dans la boîte de texte.  Si ce n'est pas le cas, essayez de copier et de coller à nouveau.

 Sauf si vous exécutez une installation portable de Beyond Compare, vous pouvez cocher la case Enregistrer pour tous les utilisateurs pour stocker la clé de licence dans le dossier des programmes, ainsi tous les utilisateurs de cet ordinateurs seront enregistrés.
