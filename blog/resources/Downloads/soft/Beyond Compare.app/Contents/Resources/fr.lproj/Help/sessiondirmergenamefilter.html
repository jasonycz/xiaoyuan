Paramètres de filtre de nom de la Fusion de Dossiers

Session > Paramètres de session > Filtres sur le nom

Vous pouvez inclure ou exclure des dossiers et des fichiers en se basant sur leurs noms en utilisant un ou plusieurs masques de fichier.

Pour définir un filtre de nom

Ajoutez un masque de fichier à la liste appropriée.  Chaque fichier doit être seul sur sa propre ligne.

•Inclure les fichiers  Fichiers à inclure dans cette session.
•Exclure les fichiers  Fichiers à exclure dans cette session.
•Inclure les dossiers  Dossiers à inclure dans cette session.
•Exclure les dossiers  Dossiers à exclure dans cette session.

Pour des exemples, voir Masques de fichier.

Pour effacer tous les filtres de nom

Cliquez sur le bouton Effacer pour effacer tous les paramètres de filtre dans cet onglet.

Pour définir le filtre de nom courant comme une prédéfinition

Cliquez sur le bouton Ajouter aux prédéfinitions, optionnellement saisissez un Nom, optionnellement modifiez le Masque et cliquez sur OK.
