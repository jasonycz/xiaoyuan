Modifier un élément Ouvrir avec

Entrez un nom dans le champ Description pour qu'il apparaisse dans le menu Ouvrir avec.

Pour définir un raccourci clavier pour l'application, déplacez-vous dans le champ Raccourci et tapez la touche souhaitée.  cliquez sur le bouton Enlever pour enlever le raccourci attribué.

Entrez une spécification appropriée de Ligne de commande pour exécuter l'application.  Elle peut inclure des commutateurs appropriés pour cette application.  Les variables suivantes seront remplacées par les informations indiquées :

%f

Chemin absolu et nom de fichier

%l

(L minuscule) Numéro de la ligne active dans une vue de fichiers

%n

Nom de fichier uniquement

%p

Chemin absolu

%x

Extension (inclut ".")

%b

Nom de fichier sans extension

%F

Le nom et le chemin relatif au dossier de travail

%P

Le chemin relatif au dossier de travail

"1" et "2" peuvent être ajoutés à la fin des variables pour se référer au fichier de gauche/droite ou au premier/deuxième fichier (ex. "BCompare %F2 %F1").  Dans tous les cas, excepté pour le deuxième %f ou %l (pour des raisons de compatibilité descendante), omettre un nombre réfère au premier fichier sans prendre en compte le nombre de fois qu'il apparaît dans la ligne de commande.

Le champ Dossier de travail vous permet de modifier où le programme s'exécute (le dossier "Démarrer dans").  S'il est vide, le programme est exécuté dans le même dossier que Beyond Compare.  Vous pouvez spécifier un dossier spécifique, ou fournir les valeurs spéciales <dossier parent> et <dossier de base> de la liste déroulante.  <dossier parent> utilisera le dossier parent du fichier sélectionné, tandis que <dossier de base> utilisera le dossier de base du côté sélectionné (ou le dossier parent uniquement dans une vue de fichiers).

Utilisez le champ Délimiteur de chemin pour remplacer tous les délimiteurs de chemins par la chaîne de caractères donnée.  Si laissé à blanc, les délimiteurs normaux seront utilisés (ex. antislash (\) sous Windows).

Les cases à cocher Accepte des fichiers et Accepte des dossiers contrôlent quand l'application est disponible pour les fichiers, les dossiers ou les deux.  Astuce : Vous pouvez configurer Beyond Compare lui-même comme une application additionnelle pour les dossiers.  De cette façon, vous pourrez sélectionner des dossiers et les comparer dans une autre instance de Beyond Compare.

Rafraîchir lorsque terminé exécute un Rafraîchissement rapide lorsque Beyond Compare a terminé avec la liste Ouvrir avec.

Lorsque la case Instances multiples est cochée, vous pouvez sélectionner de multiples fichiers ou des couples de fichiers, et exécuter l'opération sur chacun d'entre eux, ou par couple de fichiers alignés.  Avec cette option, Beyond Compare teste si un second fichier est indiqué sur la ligne de commande (ex. "%x2").  Si c'est le cas, l'application s'exécutera une fois avec chaque couple, autrement, il séparera le couple et s'exécutera pour chaque fichier sélectionné dans le couple.

Attendre qu'une instance se termine indique à Beyond Compare d'attendre que le programme précédent se termine avant de lancer la commande suivante.
