Options pour les astuces

Outils > Options > Astuces

Les astuces sont des options avancées ou obscures du programme que nous regroupons ici pour éviter d'encombrer les autres domaines.

Vérifier les mises à jour

Par défaut, Beyond Compare vérifie notre site web pour détecter s'il existe une nouvelle version de l'application.  Vous pouvez changer le nombre de jours entre les tests, ou décochez la case Tester chaque pour ne jamais tester.

Affichage de l'éditeur

La plupart des paramètres d'affichage pour les vues de texte sont trouvés sur la page Couleurs et fontes de la vue Fichiers du dialogue Options.  Les paramètres spécifiques aux modifications de texte sont ici.

Le surlignage syntaxique est normalement supprimé sur des lignes différentes afin qu'il n'interfère pas avec les schémas de coloration de différence.  Cochez la case Afficher Affiche ou cache le surlignage syntaxique sur différentes lignes pour modifier ce comportement.

Utiliser une hachure de croix pour les lignes au-delà de la fin du fichier aide à identifier la fin de vos fichiers texte, particulièrement lors de la comparaison d'un long fichier avec un plus court.

Marge sur le côté droit pour l'éditeur de gauche (comparaison uniquement) positionne les boutons de copie tout près du centre de la vue, et conserve les barres de défilement sur les bords extérieurs.  Cela n'a pas d'effet si vous utilisez l'Affichage > Disposition dessus dessous .

Activer la liaison de police dans les éditeurs permet à l'éditeur de texte d'utiliser des fontes additionnelles pour supporter des caractères de multiples langages.

Utiliser la couleur des orphelins fait que les vues de fichiers utiliseront des couleurs configurables séparément qui ne sont que d'un côté de la comparaison.

Vous pouvez ajouter ou soustraire des pixels de l'Espacement supplémentaire de ligne pour modifier la densité de texte dans la vue.

Pour afficher une ligne verticale à une colonne spécifique, définissez Afficher une ligne en colonne à la valeur souhaitée.  Définissez-la à zéro (0) pour cacher la ligne.  Si vous utilisez une police d'espacement proportionnel, la position de la colonne sera déterminée par la largeur du caractère espace.

Estomper le panneau inactif de assombrit le côté non actif de la comparaison en vue de vous aider à vous concentrer sur le côté actif.

Pour utiliser une police alternative pour des panneaux de saisie plus étroits, cochez la case Utiliser une police différente pour les panneaux en entrée de la fusion.

Opérations sur fichier

Bip après de longues opérations sur les fichiers causes un son audible lorsque des opérations de fichiers telles que la copie ou le déplacement sont terminées, sauf si elles prennent moins de 3 secondes.

L'attribut Lecture seule est normalement préservé lors de la copie de fichiers.  Sur un média en lecture seule tel que les CD ou les DVD, tous les fichiers seront encore marqués lecture seule lorsqu'ils sont copiés vers un autre endroit.  Pour éviter ça, cochez la case Enlever l'indicateur lecture seule lors de la copie d'un CD ou DVD.

Cochez Vérifier si les fichiers ont été changés sur le disque pour tester l'existence d'une nouvelle copie du fichier lorsqu'on revient à une vue de fichiers.  Si trouvé, Beyond Compare demandera "Fichier changé sur le disque, recharger ?"  Cochez Recharger automatiquement sauf si des changements doivent être ignorés pour que Beyond Compare charge automatiquement le nouveau fichier tant que vous n'avez fait aucun changement dans le fichier affiché.

La Priorité de comparaison peut être modifiée pour jouer sur l'ordre dans lequel le travail en tâche de fond sera exécuté.

Taille du tampon pour la comparaison binaire peut être modifié pour jouer sur la vitesse.

Vue Accueil

Mémoriser la dernière session de l'Accueil peut être coché pour se souvenir de la dernière session sélectionnée dans la vue Accueil au lieu de la dernière session mise à jour dans l'application.

Interface

Voici à quoi ressemble une barre d'outils lorsque les case Afficher les conseils pour les boutons de la barre d'outils, etc. et Afficher le raccourci clavier dans les conseils sont cochées :

La touche Échap ferme les vues de fichiers s'applique à tous les types de vue Fichiers tels que la Comparaison de Textes, la Comparaison de Tableaux et la Comparaison d'Images.  Elle n'affecte pas les vues Dossiers.

Ascenseur vertical simple peut être coché pour épargner de l'espace horizontal en cachant un ascenseur vertical redondant.

Mémoriser la dernière position du séparateur peut être coché pour que la taille des panneaux des vues de fichiers soit définie par l'utilisateur au lieu d'être déterminée automatiquement.

Sélectionnez Compresser pour adapter pour compresser les miniatures dans l'espace vertical disponible.  Sélectionnez Autoriser le défilement pour montrer un ascenseur vertical à la place.

Panneau du journal (Log)

Résumé des opérations

Journaliser un résumé de chaque opération.

Détails des opérations

Journaliser les détails de chaque opération, en incluant chaque fichier affecté.

Avertissements

Journal des avertissements et des erreurs.

Trace FTP

Messages de journal vers et à partir d'un site FTP.

Messages de débogage

Messages de journal utiles à l'équipe de support technique de Scooter Software.

Afficher jusqu'à définit combien de messages le journal peut gérer avant de supprimer les plus vieux.

Si un fichier est saisi pour Enregistrer automatiquement les messages dans un fichier, il sera créé lorsque ce sera nécessaire.

Métriques

Sessions enregistrées automatiquement au maximum

Définit combien de sessions sont automatiquement sauvegardées avant que la plus ancienne ne soit automatiquement supprimée.  Entrez zéro (0) pour désactiver la fonctionnalité d'enregistrement automatique des sessions.

Éléments d'utilisation la plus récente au maximum

Définit combien d'éléments peuvent être automatiquement mémorisés avant le plus ancien ne soit automatiquement supprimé.  Par exemple, ceci affecte comment la liste déroulante du dossier de base s'affiche.

Millisecondes avant que le conseil d'aide n'apparaisse

Définit combien de temps attendre avant que le popup du conseil ne soit affiché.

Millisecondes avant que le conseil d'aide ne disparaisse

Définit combien de temps le popup du conseil restera visible.

Millisecondes avant que la recherche incrémentale ne se réinitialise

Définit combien de temps attendre avant que l'utilisateur ne tape le caractère suivant pour une recherche incrémentale.

Filtres de nom prédéfinis

Spécifiez les choix des filtres de nom prédéfinis disponibles dans la barre d'outils de la vue de dossiers.  Utilisez des définitions simples de caractère générique séparées par des points-virgules pour le masque.  Exemple :

*.htm;*.html;*.jpg

 

Scripts

Cochez la case Bip quand terminé pour un signal audible lorsque le script est terminé.

 

Cochez la case Fermer quand terminé pour sortir du programme lorsque le script est terminé.

Spécifiez un Dossier des scripts partagés qui sera l'endroit pour les scripts partagés.  Si le fichier de script n'est pas trouvé dans le dossier actuel, Beyond Compare le cherchera là.

Fichier de sessions partagées

Si un Fichier de sessions partagées est fourni, les sessions partagées apparaîtront dans votre arborescence de sessions.  Vous pouvez les utiliser mais pas les modifier.  Utilisez Outils > Exporter les paramètres pour créer un fichier des sessions partagées.
