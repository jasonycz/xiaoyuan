Paramètres des autres filtres de la Fusion de Dossiers

Session > Paramètres de session > Autres filtres

Vous pouvez exclure des fichiers d'une session de dossiers en vous basant sur des critères autres que leurs noms.  Cliquez sur le bouton Nouveau pour ajouter un Élément Autre filtre.

Note  Chaque fois que d'autres éléments de filtre sont définis, une barre de commentaire apparait dans le bas de la vue de dossiers.  Vous pouvez cliquer sur le bouton Autres filtres sur elle pour modifier ces filtres.

Note  Si un filtre de fichier détermine qu'un fichier d'un côté doit être caché mais que le fichier correspondant de l'autre côté doit être visible, les deux sont rendus visibles et celui qui aurait dû être caché sera coloré en turquoise (bleu-vert).

La case Exclure les fichiers protégés du système d'exploitation est cochée par défaut.
