Restauration des défauts usine

Outils > Restaurer les défauts usine

Vous pouvez restaurer les paramètres à leurs valeurs par défaut usine.

Pour restaurer les paramètres

1.Sélectionnez Outils > Restaurer les défauts usine.
2.Cochez les cases des types de paramètres que vous souhaitez restaurer :

Options de programme inclut la plupart des préférences que vous avez définies dans le dialogue Options.

Couleurs, Fontes inclut toutes vos préférences définies dans la page correspondante du dialogue Options.

Barre d'outils, Raccourcis, Menus inclut vos préférences définies dans la page correspondante du dialogue Options.

État du programme inclut par exemple les positions des fenêtres et les listes des éléments les plus récemment utilisés (Most Recently Used).

Sessions inclut certaines ou toutes vos sessions nommées.

Formats de fichier inclut certains ou tous vos formats personnalisés de fichier.

Profils incluent certains ou tous vos profils nommés, contenant les paramètres pour FTP, etc.

3.Si vous restaurez des sessions, des formats de fichier ou des profils, cliquez sur Suivant pour sélectionner les éléments nommés à inclure.

Cochez la case Supprimer toutes les sessions existantes pour effacer toutes vos sessions stockées avant d'importer celles du fichier avant d'importer celles-là dans le fichier.  De la même façon, cochez Supprimer tous les formats de fichier existants ou Supprimer tous les profils existants si vous voulez.

4.Cliquez sur Finir pour restaurer les paramètres.
