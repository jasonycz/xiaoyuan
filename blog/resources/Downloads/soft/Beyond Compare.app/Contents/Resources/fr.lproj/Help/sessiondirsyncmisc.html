Paramètres divers de Synchronisation de Dossiers

Session > Paramètres de session > Divers

Pour forcer un alignement différent

Par défaut, une session de dossiers aligne les fichiers et les sous-dossiers par nom, à gauche et à droite.  Cliquez sur le bouton Nouveau pour ajouter une Substitution d'alignement.

Note  La façon la plus facile d'aligner manuellement deux éléments dans une session de dossiers est de faire un clic-droit sur le premier élément, choisir Aligner avec, et cliquer sur le second élément.

Pour substituer des associations de format de fichier

La liste Formats de fichier activés vous permet de substituer les formats de fichiers qui seront utilisés dans une session de dossiers.  Un format de fichier est listé en gras si son état d'activation dans cette session est différent de son état d'activation global.  Vous pouvez créer un format de fichier personnalisé qui est activé pour son utilisation dans cette session mais désactivé dans la liste globale.
