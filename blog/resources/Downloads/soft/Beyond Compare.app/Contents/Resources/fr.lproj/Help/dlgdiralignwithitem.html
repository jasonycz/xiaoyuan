Modifier une substitution d'alignement

Remplissez les champs Aligner le fichier (ou dossier) de gauche et Aligner le fichier (ou dossier) de droite.

Cochez la case Expression régulière pour les interpréter comme des expressions régulières comme style de remplacement.  Si vous utilisez des caractères génériques DOS tels que * et ?, laissez-le décoché.

Remplissez le champ Limiter à ce dossier pour spécifier un chemin relatif pour l'alignement.

Exemples

Aligner le fichier (ou dossier) de gauche

avec le fichier (ou dossier) de droite

Description

x.txt

x.csv

Aligne "x.txt" avec "x.csv".

*.gif

*.png

Aligne tous les fichiers GIF avec les fichiers PNG de même nom.

x*.jpg

y*.jpg

Aligne des fichiers tels que "x1.jpg" avec "y1.jpg" et "x2.jpg" avec "y2.jpg".
