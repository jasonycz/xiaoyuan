Instantanés

Dans le menu Outils, la commande Enregistrer un Instantané enregistre une liste compressée en lecture seule du dossier spécifié et de tous ses sous-dossiers, incluant l'horodatage, la taille et les attributs de chaque fichier.  Les Instantanés ont .bcss comme extension de fichier.

Les Instantanés sont utiles pour détecter les changements au cours du temps.  Les fichiers Instantanés peuvent être chargés à une date ultérieure et comparés à un dossier courant.  Vous pouvez charger un Instantané exactement comme tout autre dossier de base.  Les comparaisons binaires et basées sur des règles ne fonctionnent pas sur les Instantanés puisque les fichiers dans les Instantanés n'existent pas réellement.

Si les CRC des fichiers sont inclus dans l'Instantané, alors les comparaisons de CRC peuvent être utilisées pour déterminer si les contenus des fichiers ont changé.  Ceci peut être utile pour détecter des corruptions de données et d'autres changements subtils de vos données.

Un Instantané se souvient du nom du dossier de base d'où il a été produit.  Si vous chargez un Instantané sur une ligne de commande (ou déposez un fichier Instantané sur l'icône Beyond Compare), le programme comparera automatiquement l'Instantané avec le dossier original.
