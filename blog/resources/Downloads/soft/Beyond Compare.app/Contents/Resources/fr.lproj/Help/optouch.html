Horodater

Actions > Horodater

Dans le menu Actions, la commande Horodater modifie l'horodatage (date/heure de dernière modification) des fichiers sélectionnés.

Si Copier les horodatages à partir du côté opposé est coché, alors les horodatages des fichiers sélectionnés seront définis pour correspondre à ceux de l'autre côté.  Si Définir l'horodatage à --> est utilisé, alors n'importe quelles date/heure peuvent être sélectionnées du calendrier.

La sélection Défaut contrôle comment le calendrier est initialisé.  Cochez Date/heure système pour utiliser la date et l'heure du système.  Cochez Dernière modification pour utiliser l'horodatage de la dernière opération d'horodatage.  La sélection de Défaut réinitialise uniquement le calendrier -- vous pouvez encore le changer pour n'importe quelle date ou heure.

Si un dossier est sélectionné, la commande Horodater essaiera de changer l'horodatage du dossier mais pas celui de son contenu.  Cette commande échouera si le système d'exploitation n'autorise pas les changements d'horodatage de dossier.
