Options de démarrage

Lorsque Beyond Compare démarre, il affiche normalement la vue Accueil.  Pour que le programme démarre automatiquement avec les mêmes sessions qui étaient ouvertes la dernière fois que vous avez utilisé Beyond Compare, spécifiez le même nom d'espace de travail dans les champs Charger l'espace de travail et À la sortie, enregistrer sous.  Dans le menu Sessions, vous pouvez également utiliser les commandes Enregistrer l'espace de travail sous et Charger l'espace de travail pour davantage de contrôle.

Lorsque vous démarrer Beyond Compare avec un couple de fichiers spécifiés, le programme peut afficher un dialogue avec un résumé de la comparaison, plutôt que de prendre le temps d'afficher la comparaison complète côte à côte.  Pour préfacer une comparaison de fichier avec le dialogue Comparaison rapide :

1.Cochez la case Afficher le dialogue de comparaison rapide en démarrant une comparaison de fichiers.
2.Sélectionnez Comparaison rapide binaire pour exécuter une comparaison octet par octet, ou Comparaison rapide basée sur des règles pour utiliser les paramètres de format de fichier appropriés aux fichiers spécifiés.
3.Cochez Ouvrir automatiquement une vue si les fichiers sont différents si vous voulez que le dialogue apparaisse seulement quand les fichiers correspondent.

Pour ajouter Beyond Compare au menu clic-droit de l'Explorateur

Cochez Inclure Beyond Compare dans le menu contextuel de l'Explorateur.

Sélectionnez n'importe quelle commande facultative listée dans le tableau, et choisissez Menu principal ou Sous-menu pour l'inclure dans le menu contextuel.  Cochez Comparer en utilisant pour ajouter un sous-menu qui liste les affichages spécifiques tels que la vue Comparaison de Tableaux ou la vue Comparaison Hexadécimale.  Ceci vous permet de supplanter la vue qui aurait été normalement sélectionnée via la liste des formats de fichier.  Cochez Fusionner pour ajouter le support de la fusion 3-voies dans le menu contextuel.  Cochez Synchroniser pour supporter la synchronisation des dossiers à partir de l'Explorateur.  Cochez Modifier pour supporter la modification d'un seul fichier à partir de l'Explorateur.  Cochez Comparer au registre pour supporter la comparaison d'un fichier registre au registre actif.
