Paramètres de grammaire de format de texte

Format Texte > Grammaire

L'onglet Grammaire définit la syntaxe d'un fichier.  Les éléments de grammaire tels que les commentaires et les mots-clés sont définis ici.  Chaque élément de grammaire peut être décrit pour un ou plusieurs éléments de grammaire qui apparaissent dans la liste.  L'ordre de la liste est significatif.  Les éléments qui sont plus hauts dans la liste sont prioritaires sur les éléments plus bas.  Cliquez sur le bouton Nouveau pour ajouter un Élément de grammaire.

Les Poids de ligne sont utilisés pour améliorer l'alignement dans une comparaison.  L'algorithme essaiera d'aligner des lignes correspondantes avec des poids de ligne plus grands.
