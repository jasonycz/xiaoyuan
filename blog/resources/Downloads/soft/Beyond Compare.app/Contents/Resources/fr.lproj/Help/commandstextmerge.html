Commandes de Fusion de Textes

Ce qui suit est une description des commandes de Fusion de Textes qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de la Fusion de Textes.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Fusion de Textes vide.

Permuter les côtés

Échange les fichiers de gauche et de droite.

Recharger les fichiers

Recharge les fichiers affichés en demandant de les enregistrer avant si cela est nécessaire.

Recomparer les fichiers

Recompare les fichiers affichés, en préservant les modifications, mais pas les possibilités de retour arrière.

Info de Fusion de Textes

Affiche les statistiques de la fusion.

Comparer à la sortie >

  Côté gauche / Partie centrale / Côté droit

Ouvre le fichier du côté indiqué et le fichier en sortie dans une nouvelle session de Comparaison de Textes.

Fusionner les dossiers parents

Ouvre une nouvelle session de Fusion de Dossiers basée sur les dossiers parents des dossiers de base actuels.

Le menu Fichier

Ouvrir un fichier

Ouvre un fichier existant dans le panneau sélectionné.

Ouvrir un fichier du système de fichier *

Ouvre un fichier local existant dans le panneau sélectionné.

Ouvrir un fichier en utilisant un profil *

Ouvre un fichier existant dans le panneau sélectionné en utilisant un profil.

Ouvrir le presse-papiers

Charge le contenu du presse-papiers dans le panneau sélectionné.

Ouvrir avec

Ouvre le fichier du panneau sélectionné avec une des applications externes Ouvrir avec.

Ouvrir avec >

  Application associée

Ouvre le fichier du panneau sélectionné avec son application associée.

Ouvrir avec >

  Éditeur de texte

Ouvre le fichier du panneau sélectionné avec l'éditeur de texte intégré.

Ouvrir avec >

  Patch de texte

Ouvre le fichier du panneau sélectionné avec la vue de patch intégrée.

Enregistrer le fichier

Sauvegarde le fichier modifié du panneau sélectionné.

Enregistrer le fichier sous

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom.

Enregistrer le fichier dans le système de fichier sous *

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom sur le système de fichier.

Enregistrer le fichier en utilisant le profil sous *

Sauvegarde le fichier du panneau sélectionné sous un nouveau nom en utilisant un profil.

Explorateur

Affiche le menu contextuel du gestionnaire de fichier pour le fichier du panneau sélectionné.

Le menu Edition

Défaire

Défait l'action précédente.

Refaire

Refait une action précédemment défaite.

Aligner avec

Vous permet de sélectionner une ligne, et ensuite de cliquer sur une deuxième ligne pour les forcer à être côte à côte.

Isoler

Réaligne la comparaison de telle sorte que les lignes sélectionnées restent seules avec elles-mêmes.

Conflit

Efface ou définit un conflit pour les lignes sélectionnées ou la section active.

Prendre à gauche / Prendre au centre / Prendre à droite

Prend les lignes sélectionnées ou la section active du côté indiqué.

Prendre à gauche puis à droite / Prendre à droite puis à gauche

Prend les lignes sélectionnées ou la section active de la gauche de de la droite dans l'ordre indiqué.

Prendre la ligne de gauche / Prendre la ligne du centre / Prendre la ligne de droite

Prend la ligne active du côté indiqué.

Augmenter l'indentation

Augmente l'indentation des lignes sélectionnées.

Diminuer l'indentation

Diminue l'indentation des lignes sélectionnées.

Couper / Copier / Coller / Supprimer

Commandes d'édition standard.

Supprimer une ligne *

Supprime la ligne active.

Supprimer jusqu'au début de ligne *

Supprime le texte de la position actuelle jusqu'au début de la ligne.

Supprimer jusqu'à la fin de ligne *

Supprime le texte de la position actuelle jusqu'à la fin de la ligne.

Supprimer un mot *

Supprime le mot à la position du curseur.

Supprimer jusqu'au début du mot *

Supprime à partir de la position actuelle jusqu'au début du mot.

Supprimer jusqu'à la fin du mot *

Supprime à partir de la position actuelle jusqu'à la fin du mot.

Insérer une ligne avant *

Insère une nouvelle ligne vide avant la ligne active.

Insérer une ligne après *

Insère une nouvelle ligne vide après la ligne courante.

Sélectionner tout

Sélectionne toutes les lignes visibles dans le panneau actif.

Sélectionner une section

Sélectionne toutes les lignes dans la section active.

Comparer la sélection au presse-papiers

Compare le texte sélectionné au contenu du presse-papiers dans une nouvelle vue.

Modification plein écran

Bascule entre le mode caractère d'édition complète (coché) et le mode ligne comme Beyond Compare 2 (décoché) dans les panneaux principaux de fichier.

Fichiers différents précédents

Prendre le fichier de droite et ouvrir la prochaine différence

Prend le fichier entier du côté droit et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Prise de gauche suivante / Prise de droite suivante

Se positionne sur la plage suivante de lignes où le côté indiqué a été pris.

Prise de gauche précédente / Prise de droite précédente

Se positionne sur la plage précédente de lignes où le côté indiqué a été pris.

Modification suivante

Se positionne sur la ligne modifiée suivante, après le curseur.

Modification précédente

Se positionne sur la ligne modifiée précédente, avant le curseur.

Rechercher

Recherche un texte correspondant.

Remplacer

Remplace le texte correspondant.

Rechercher le suivant

Recherche l'occurrence suivante de la chaîne de caractères de recherche.

Rechercher le précédent

Recherche l'occurrence précédente de la chaîne de caractères de recherche.

Aller en

Se positionne sur la ligne et la colonne indiquées.

Basculer un signet

Place un signet numéroté sur la ligne active de la comparaison, numéroté de 0 à 9.

Aller au signet

Se repositionne au signet spécifié.

Effacer les signets

Enlève tous les signets de la comparaison.

Le menu Affichage

Afficher tout

Affiche des lignes non modifiées, avec une modification identique, une modification à gauche, une modification à droite, une modification différente et en conflit.

Afficher les modifications

Affiche des lignes avec une modification identique, une modification à gauche, une modification à droite, une modification différente et en conflit.

Afficher les conflits

Affiche les lignes en conflit.

Afficher les modifications de gauche

Affiche des lignes avec une modification identique, une modification à gauche, une modification différente et en conflit.

Afficher les modifications de droite

Affiche des lignes avec une modification identique, une modification à droite, une modification différente et en conflit.

Afficher ce qui est fusionnable

Affiche des lignes avec une modification identique, une modification à gauche, une modification à droite et une modification différente.

Afficher les inchangés

Affiche les lignes non modifiées.

Afficher aucun *

N'affiche aucune ligne.

Afficher le contexte

Affiche les lignes environnantes des différences.  Le nombre des lignes de contexte peut être défini dans les Options de Vue Texte.

Ignorer les différences non importantes

Traite les différences non importantes comme identiques.

Ignorer les modifications identiques

Traite des modifications identiques comme identiques.

Favoriser les modifications de gauche

Supprime le surlignage des modifications uniquement à gauche dans la sortie.

Favoriser les modifications de droite

Supprime le surlignage des modifications uniquement à droite dans la sortie.

Ignoré

Ignore (ou prend en compte) les différences dans les lignes sélectionnées ou dans la section active.

Espace blanc visible

Affiche ou cache les espaces, tabulations.

Numéros de ligne

Affiche ou cache les numéros de ligne.

Surlignage syntaxique

Affiche ou cache le surlignage syntaxique.

Police d'affichage *

Sélectionne la police à utiliser dans les panneaux d'édition.

Disposition côte à côte

Arrange les panneaux de l'éditeur à gauche et à droite l'un de l'autre.

Disposition dessus dessous

Arrange les panneaux de l'éditeur au-dessus en au-dessous l'un de l'autre.

Panneau de sortie détaché

Affiche le panneau de sortie dans une fenêtre séparée.

Pages web

Affiche ou cache les panneaux du navigateur web qui affichent les fichiers en cours comme des pages web.

Miniatures

Affiche ou cache le résumé miniature des changements à gauche de la vue.

Panneau central

Affiche ou cache le panneau central.

Détails en texte

Affiche ou cache le panneau des détails en texte en bas de la vue.

Détails en hexadécimal

Affiche ou cache le panneau des détails en hexadécimal en bas de la vue.

Règle

Affiche ou cache la règle des détails de ligne.

Informations de fichier

Affiche ou cache les panneaux d'information de fichier au-dessus des panneaux d'édition.

Barre d'outils

Affiche ou cache la barre d'outils.
