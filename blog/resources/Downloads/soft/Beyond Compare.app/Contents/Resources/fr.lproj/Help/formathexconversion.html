Paramètres de conversion de Format hexadécimal

Format Hexadécimal > Conversion

L'onglet Conversion définit comment les fichiers sont chargés et enregistrés.

Sélectionnez une méthode de Conversion.  Lors de l'utilisation de méthodes Programme externe vous pouvez spécifier un programme de conversion avec le chemin d'accès approprié et les commutateurs de ligne de commande dans le champ Chargement.  Afin de supporter les noms de fichier avec des caractères étendus, soyez certain de bien choisir l'encodage du nom de fichier (Unicode ou ANSI).  Les variables suivantes seront remplacées par les informations indiquées :

 %s        fichier source

 %t        fichier cible

 %o        fichier original

On considère que les programmes de conversion se terminent avec succès s'ils retournent un code de sortie égal à zéro, et s'ils génèrent un fichier non vide.

Si vous avez un programme de conversion que vous voulez utiliser avant l'enregistrement, vous pouvez décochez la case Désactiver les modifications et dans le champ Enregistrement, spécifiez le chemin d'accès de ce programme de conversion et des commutateurs de ligne de commande.  Les variables ci-dessus s'appliquent.
