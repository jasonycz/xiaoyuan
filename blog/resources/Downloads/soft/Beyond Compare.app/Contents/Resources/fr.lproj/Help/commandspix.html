Commandes de Comparaison d'Images

Ce qui suit est une description des commandes de Comparaison d'Images qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de la Comparaison d'Images.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Comparaison d'Images vide.

Permuter les côtés

Échange les fichiers de gauche et de droite.

Recharger les fichiers

Recharge les fichiers affichés.

Recomparer les fichiers

Recompare les fichiers affichés sans rechargement.

Rapport de Comparaison d'Images

Produit le Rapport de Comparaison d'Images.

Informations de Comparaison d'Images

Affiche les statistiques de comparaison.

Comparer des fichiers en utilisant

Ouvre les fichiers affichés dans une nouvelle session en utilisant une classe différente de vue de fichiers.

Comparer les dossiers parents

Ouvre une nouvelle session de Comparaison de Dossiers basée sur les dossiers parents des fichiers actuellement chargés.

Le menu Fichier

Ouvrir un fichier

Ouvre un fichier existant dans le panneau sélectionné.

Ouvrir un fichier du système de fichier *

Ouvre un fichier local existant dans le panneau sélectionné.

Ouvrir un fichier en utilisant un profil *

Ouvre un fichier existant dans le panneau sélectionné en utilisant un profil.

Ouvrir le presse-papiers

Charge le contenu du presse-papiers dans le panneau sélectionné.

Ouvrir avec

Ouvre le fichier du panneau sélectionné avec une des applications externes Ouvrir avec.

Ouvrir avec >

  Application associée

Ouvre le fichier du panneau sélectionné avec son application associée.

Explorateur

Affiche le menu contextuel du gestionnaire de fichier pour le fichier du panneau sélectionné.

Le menu Edition

Copier

Copie les lignes sélectionnées dans le presse-papiers.

Le menu Recherche

Fichiers différents suivants

Ouvre le couple suivant de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Fichiers différents précédents

Ouvre le couple précédent de fichiers avec des différences, fichiers de la session du dossier parent.  (Session filles uniquement.)

Copier le fichier vers la droite et ouvrir le différent suivant

Copie le fichier entier du côté gauche vers la droite et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Copier le fichier vers la gauche et ouvrir le différent suivant

Copie le fichier entier du côté droit vers la gauche et ouvre les prochains fichiers avec différences de la session de dossiers.  (Session filles uniquement.)

Le menu Affichage

Mode Tolérance

Détecte les différences de pixel plus grandes que la tolérance configurable.

Mode Plage de non-correspondance

Mesure le degré de différence de pixels.

Mode Fusion

Incruste les images en se basant sur un pourcentage configurable.

Ignorer les différences non importantes

Traite les différences non importantes comme identiques.

Taille automatique

Force les images à avoir la même hauteur ou largeur.

Pivoter Sens horlogique

Fait tourner l'image actuelle de 90 degrés vers la droite.

Pivoter Sens anti-horlogique

Fait tourner l'image actuelle de 90 degrés vers la gauche.

Retourner horizontalement

Échange les pixels de l'image active relativement à son centre horizontal.

Retourner verticalement

Échange les pixels de l'image active relativement à son centre vertical.

Réinitialiser le décalage de différence

Réaligne le coin supérieur gauche des deux images.

Bascule l'incrustation

Permute entre les images de droite et de gauche.

Informations de fichier

Affiche ou cache les panneaux d'information de fichier au-dessus des panneaux.

Plein écran

Affiche le panneau sélectionné en plein écran.  Appuyez sur ESC pour sortir du mode plein écran.

Zoom  >

  Agrandir

Fait en sorte que les images apparaissent plus grandes.

Zoom  >

  Réduire

Fait en sorte que les images apparaissent plus petites.

Zoom  >

  Taille réelle

Fait en sorte que les images apparaissent en taille réelle.

Zoom  >

  Ajuster à la fenêtre

Fait en sorte que les images remplissent les panneaux.

Barre d'outils

Affiche ou cache la barre d'outils.

* Caché par défaut.
