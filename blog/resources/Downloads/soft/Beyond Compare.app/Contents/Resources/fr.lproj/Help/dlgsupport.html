Rapporter des problèmes

Aide > Support

Si vous avez découvert un problème avec Beyond Compare, nous souhaiterions que vous nous en parliez !

À partir de n'importe quelle fenêtre de niveau supérieur, sélectionnez &Aide > Support, et suivez les instructions pour produire un package contenant les paramètres de votre programme et du système.  Attachez le package à un courriel en y incluant une description du problème, et envoyez-le nous à support@scootersoftware.com.

Vous pouvez souhaiter inclure une copie d'écran qui illustre le problème.  Si c'est le cas, suivez ces étapes pour créer un courriel assez petit pour envoyer :

1.Appuyez sur Alt+ImpÉcr pour placer une copie de la fenêtre active dans le presse-papiers.
2.Ouvrez le programme Paint de Microsoft (Démarrer > Programmes > Accessoires > Paint) et presser Ctrl+V pour coller l'image.
3.Sélectionnez la partie de l'image que vous voulez envoyer.
4.Sélectionnez Edition > Copier vers, changez Enregistrer sous le type en soit 16 couleurs ou 256 couleurs, puis entrez un nom de fichier.  (Si les types PNG et GIF ne sont pas disponibles, sélectionnez Bitmap 256 couleurs et compressez le fichier image avec un utilitaire zip.)
5.Joignez le fichier image au courrier électronique que vous nous destinez.
