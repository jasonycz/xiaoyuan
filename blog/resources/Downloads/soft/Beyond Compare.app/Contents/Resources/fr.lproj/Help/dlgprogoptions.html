Options du programme

Outils > Options

Pour définir les préférences du programme

1.Sélectionnez Outils > Options pour ouvrir le dialogue Options.
2.Dans la liste de gauche, cliquez sur l'élément contenant les options que vous souhaitez définir :

Démarrage

Onglets

Couleurs, Fontes

 Vues Dossiers

 Vues Fichiers

 Comparaison d'Images

Modification de texte

Différence suivante

Sauvegardes

Opérations sur fichier

Types d'archive

Barre d'outils, etc.

Ouvrir avec

Astuces

3.Modifiez les paramètres souhaités et ensuite cliquez sur le bouton Appliquer pour accepter les modifications, ou sur le bouton OK pour appliquer et fermer.
