Modifier un élément Texte de remplacement

Spécifiez le Texte à rechercher et le texte Remplacer avec.

Cochez la case Respecter la casse de caractère pour faire coïncider le texte uniquement avec les mêmes caractères, majuscules ou minuscules.

Cochez la case Mots entiers uniquement pour empêcher la correspondance avec des parties de mots.

Cochez la case Expression régulière pour les interpréter comme des expressions régulières comme style de remplacement.  Cliquez sur le bouton Insérer un élément d'expression régulière pour choisir dans la liste des termes usuels d'expression régulière.  Cliquez sur le bouton Exemples d'ER pour plus d'informations.

Spécifiez le Côté du panneau de l'éditeur qui doit être cherché.
