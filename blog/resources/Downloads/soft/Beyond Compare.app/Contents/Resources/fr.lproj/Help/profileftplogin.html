Paramètres de connexion de profil FTP

Profil FTP > Identification

Protocole

Spécifiez FTP ou une des options FTP sécurisé : SSL ou SSH.

Port

Écrasez si vous ne voulez pas utiliser les valeurs standard pour le protocole spécifié.

Hôte, Utilisateur, Mot de passe

Spécifiez les paramètres de connexion pour votre compte.

Enregistrer le mot de passe

Cochez pour chiffrer faiblement votre mot de passe et le mémoriser avec les autres paramètres du profil FTP.

La Description peut inclure n'importe quel texte que vous souhaitez.
