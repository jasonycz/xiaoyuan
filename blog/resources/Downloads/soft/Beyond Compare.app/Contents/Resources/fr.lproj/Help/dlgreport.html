Rapports

Les rapports de comparaison vous permettent d'enregistrer les informations d'une comparaison pour la visualiser plus tard sans devoir ré-exécuter la comparaison.  Ces rapports peuvent être imprimés directement de Beyond Compare ou enregistrés en HTML ou au format texte brut.

La fonctionnalité de rapport est disponible dans le menu Session des différents types de session.  Par exemple, dans une session de Comparaison de Dossiers, vous pouvez sélectionner Session > Rapport de Comparaison de Données .

Le style des rapports disponibles diffère selon les divers types de session, mais les options sont très similaires.  Vous pouvez sortir un rapport vers n'importe quelle de ces destinations :

•Imprimante
•Rapport HTML (fichier ou presse-papiers)
•Texte brut (fichier ou presse-papiers)

 

Les choix Imprimante et HTML supportent le surlignage des différences en couleur.

Vous pouvez prévisualiser un rapport d'Imprimante avec lae bouton Aperçu de l'impression, ou un rapport HTML, avec la bouton Visualiser dans un navigateur.
