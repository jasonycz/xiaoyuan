Parcourir les différences

Dans le menu Recherche, les commandes Section de différences suivante et Section de différences précédente vous aident à parcourir toutes les différences dans vos fichiers, et tous vos fichiers avec différences.  Dans la Comparaison de Textes, sélectionnez Section de différences suivante pour vous positionner à l'alignement suivant de lignes avec des différences, ou sur Section de différences précédente pour retourner à l'alignement précédent de lignes avec des différences.

Les icônes de flèche seront désactivées ou changeront de couleur du jaune clair au jaune terne pour indiquer qu'il n'y a plus de différences dans cette direction.  Pour configurer ce qui se passe quand il n'y a plus de différence, voir Options de prochaine différence de le vue Fichiers.

Dans les sessions de Comparaison de Textes, le format de fichier courant contrôle si Suivant et Précédent sont sur une base de ligne-par-ligne ou de section-par-section.  Pour accéder au format de fichier actif d'une session de Comparaison de Textes, sélectionnez Session > Paramètres de session , aler à l'onglet Importance, et cliquez sur le bouton Modifier la grammaire.  Ensuite, dans l'onglet Divers du dialogue Format de texte, cochez Les lignes sont indépendantes.

Dans le menu Recherche, sélectionnez Différence suivante ou Différence précédente pour vous positionner sur des différences individuelles dans de longues lignes avec de multiples différences.

Dans le menu Recherche, sélectionnez Fichiers différents suivants ou Fichiers différents précédents pour vous positionner directement sur les prochains fichiers avec des différences.
