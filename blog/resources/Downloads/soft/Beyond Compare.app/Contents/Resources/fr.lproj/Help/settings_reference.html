Référence des paramètres

Beyond Compare est hautement configurable et il peut être parfois difficile de savoir où aller dans le programme pour modifier un paramètre spécifique.  Voici où regarder :

Options du programme

Les préférences générales de l'utilisateur sont définies dans le dialogue Options.  Sélectionnez Outils > Options pour l'ouvrir.  À partir de là, vous pouvez définir les options de démarrage, contrôler les messages de confirmation, et personnaliser les couleurs d'affichage et les polices.

Paramètres de session

La plupart des options qui définissent l'étendue et les règles d'une comparaison peuvent être trouvées dans le dialogue Paramètres de session.  Chaque type de session a son propre dialogue de paramètres de session.  Sélectionnez Session > Paramètres de session pour ouvrir le dialogue, ou cliquez simplement sur le bouton Règles sur la barre d'outils.

Les paramètres de session sont stockés avec vos sessions enregistrées.

Formats de fichier

Les types spécifiques de fichier peuvent être configurés dans le dialogue Formats de fichier.  Sélectionnez Outils > Formats de fichier pour l'ouvrir.

Profils

Des services distants spécifiques peuvent être configurés dans le dialogue Profils.  Sélectionnez Outils > Profils pour l'ouvrir.

Intégration de contrôle de source

Des associations de contrôle de source spécifiques peuvent être configurées dans le dialogue Intégration du Contrôle de Source.  Sélectionnez Outils > Intégration du contrôle de source pour l'ouvrir.
