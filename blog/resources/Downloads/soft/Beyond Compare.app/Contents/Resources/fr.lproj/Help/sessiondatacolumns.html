Paramètres de colonne de Comparaison de Tableaux

Session > Paramètres de session > Colonnes

Chaque ligne de l'affichage représente une colonne de comparaison, et affiche :

•Comparaison si clé, numéro de colonne, nom de colonne
•Fichier de gauche numéro de colonne, échantillon de données si disponible
•Fichier de droite numéro de colonne, échantillon de données si disponible
•Traitement résumé de la comparaison de colonne

Pour contrôler l'alignement des lignes

Définissez autant que nécessaires des colonnes comme clés pour identifier univoquement chaque ligne.  Par défaut, la première colonne est la seule qui est définie comme une clé.  Pour définir une colonne comme une clé, positionnez-vous sur la ligne appropriée, cliquez sur le bouton Modifier, cochez la case Clé, puis cliquez sur OK.  Si de multiples clés sont définies, la priorité suit l'ordre dans la comparaison.  Si aucune clé n'est définie, l'alignement utilisera toutes les colonnes pour identifier univoquement chaque ligne.

Par défaut, les fichiers sont triés par leurs clés avant que les lignes ne soient alignées.  Cochez la case Alignement non trié pour forcer l'alignement à préserver l'ordre des lignes telles qu'elles existent dans les fichiers.  Notez que cela peut engendrer que certaines lignes avec la même donnée de colonne clé ne soient pas alignées.

Pour contrôler l'alignement des colonnes

Par défaut, la comparaison suppose que les colonnes des fichiers sont dans le même ordre.  Pour modifier l'ordre de comparaison d'une colonne du fichier de gauche, positionnez-vous sur la ligne appropriée sous Fichier de gauche, et cliquez sur le bouton Déplacer vers le haut ou Déplacer vers le bas autant que nécessaire.  Si à la place, vous vous positionnez sous Comparaison, ces boutons changeront la position de la colonne de comparaison elle-même.  Pour empêcher qu'une colonne du fichier de gauche soit comparée, positionnez-vous sous Fichier de gauche et cliquez sur le bouton Enlever.  Si à la place, vous vous positionnez sous Comparaison, ce bouton empêchera les colonnes de son fichier à être comparées et supprimera la définition de la colonne de comparaison.  Pour restaurer une colonne enlevée du fichier de gauche, positionnez-vous sous Fichier de gauche, cliquez sur le bouton Insérer, marquez l'élément dans le dialogue Insérer des colonnes et cliquez sur OK.  Cliquez sur le bouton Tasser pour enlever toutes les lignes blanches de la liste.

Pour contrôler les noms des colonnes

Par défaut, les noms de colonne sont déterminés automatiquement à partir du fichier de gauche ou de droite.  Pour donner un certain nom à une colonne, cliquez sur le bouton Modifier, changez le Nom et cliquez sur OK.  Pour revenir au nommage automatique, sélectionnez Réinitialiser les noms du menu contextuel (bouton droit de la souris) de la liste.  Si des données du fichier de gauche sont disponibles, vous pouvez sélectionner Remplir les noms en utilisant le fichier de gauche du menu contextuel de la liste pour forcer les noms à être ceux trouvés dans le fichier de gauche.  Remplir les noms en utilisant une comparaison forcera les noms à être ceux qui sont automatiquement déterminés.

Pour contrôler comment les données de colonne sont comparées

Positionnez-vous sur la ligne appropriée et cliquez sur le bouton Modifier pour ouvrir le dialogue Traitement de colonne.  Pour changer les paramètres par défaut du Traitement, cliquez sur le bouton Modifier les défauts.
