Pour substituer des associations de format de fichier

La liste Formats de fichier activés vous permet de substituer les formats de fichiers qui seront utilisés dans une session de dossiers.  Un format de fichier est listé en gras si son état d'activation dans cette session est différent de son état d'activation global.  Vous pouvez créer un format de fichier personnalisé qui est activé pour son utilisation dans cette session mais désactivé dans la liste globale.
