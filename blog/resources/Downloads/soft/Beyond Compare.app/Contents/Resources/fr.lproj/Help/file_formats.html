Formats de fichier

Beyond Compare utilise des formats de fichier pour spécifier comment certains types de fichiers doivent être pris en compte.  Par exemple, des fichiers avec une extension .jpg ou .png sont associés à la Comparaison d'Images, tandis que .c et .cpp sont reconnus comme ayant une syntaxe C/C++ et seront gérés par la Comparaison de Textes.  Vous pouvez modifier les formats de fichier prédéfinis ou créer les vôtres.

Les formats de fichier déterminent :

•Le type de vue utilisée pour une comparaison visuelle
•Quelle conversion, s'il y en a une, doit être exécutée avant de comparer ou de sauvegarder
•La syntaxe, ou la grammaire, du contenu du fichier
•La sensibilité à la casse des éléments de grammaire
•Les tabulations
•Les poids de ligne pour aider le processus d'alignement

Les formats de fichier contrôlent principalement comment les vues de fichiers gèrent vos fichiers, mais elles affectent également comment une session de dossier compare les contenus de fichiers.  Si vous utilisez une comparaison de contenu basée sur des règles, la session de dossiers peut être limitée uniquement à l'affichage des fichiers avec des différences importantes.
