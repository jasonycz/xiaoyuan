Text-Merge verwenden

Wenn Beyond Compare einen Vergleich durchgeführt hat, wird das Programm bis zu vier Bereiche anzeigen:  Den linken, mittleren und rechten Bereich im oberen Teil des Fensters, und einen Ausgabebereich im unteren Teil.  Wenn Sie einen Mittelbereich einbeziehen, ist Beyond Compare in der Lage, beide Dateien mit ihm zu vergleichen, und Änderungen als Hinzufügungen, Änderungen oder Löschungen auf den entsprechenden Seiten zu kennzeichnen.  Änderungen auf der linken Seite werden in Türkis markiert, und Änderungen auf der rechten Seite in Magenta.  Beidseitige Änderungen werden als Konflikt betrachtet, welche rot markiert werden.  (Wenn Sie es wünschen, können Sie das Farbschema ändern.)

Konflikte

Immer dann, wenn es unterschiedliche Änderungen auf der linken und auf der rechten Seite gibt, die auf derselben Zeile oder innerhalb zweier Zeilen voneinander auftreten, werden diese Abschnitte als Konflikt angesehen, der sodann in allen drei Eingabebereichen rot gekennzeichnet wird.  Die Ausgabe wird als Konflikt bezeichnet.  Wenn auf beiden Seiten (links und rechts) dieselbe Änderung vorgenommen wurde, wird dies nicht als Konflikt betrachtet.

Anzeigefilter

Beim Zusammenführen werden die traditionellen Anzeigefilter von Beyond Compare in neue gewechselt, die den Merge-Status (Zusammenführungsstatus) jeder Zeile angeben.  Die Verwendung dieser Filter ermöglicht die Einschränkung der Anzeige auf nur Konflikte, oder nur Änderungen auf der einen Seite, oder der anderen.

Zusammenführen (Merge)

Sie können sowohl in den Eingabe- als auch Ausgabebereichen von Unterschied zu Unterschied navigieren.  Dabei werden die Fenster immer synchron bleiben.  Wenn Sie auf einem Unterschied stehen, können Sie die Befehle Von links übernehmen , Von Mitte übernehmen und Von rechts übernehmen verwenden, um die Änderungen von links oder rechts zu übernehmen, oder das mittlere Original beizubehalten.  Die "Übernehmen"-Befehle stehen sowohl in der Symbolleiste zur Verfügung, als auch im Kontextmenü jedes Bereiches und als Schaltflächen an den jeweiligen Abschnitten im Ausgabebereich.  Beim ersten Laden wird der Ausgabebereich mit dem Mittelbereich übereinstimmen, außer wenn automatisch nicht-konfliktbehaftete Änderungen übernommen werden können.  Jede Änderung wird entsprechend ihres Herkunftseingabebereiches farblich gekennzeichnet.  Beim Zusammenführen können Sie den Ausgabebereich auch direkt bearbeiten, was die Farbe des Abschnittes in Gelb ändert.  Wenn Sie die Änderungen auf die ursprünglichen Werte zurückstellen möchten, verwenden Sie einfach erneut den "Übernehmen"-Befehl.
