Einstellungen importieren

Extras > Einstellungen importieren

Sie können Einstellungen importieren, die zuvor aus Beyond Compare exportiert wurden, sogar von einer älteren Version wie Beyond Compare 2.

Um Einstellungen zu importieren

1.Wählen Sie Extras > Einstellungen importieren.
2.Wählen Sie jene Datei, die die zu importierenden Einstellungen enthält.  Es kann sich um eine Einstellungenpaket-Datei von Beyond Compare (.bcpkg) oder eine XML-Datei handeln.  Es kann sich zudem um eine Exportdatei von Beyond Compare 2 handeln (ZIP- oder XML-Datei).
3.Sollte die Datei unterschiedliche Einstellungsarten enthalten, markieren Sie die entsprechenden Kontrollkästchen für die jeweils zu importierenden Einstellungen:

Programmoptionen enthalten den Großteil Ihrer im Optionendialog eingestellten Präferenzen.

Farben, Schriftarten beinhält alle Ihre Präferenzen, die Sie auf ihrer Seite im Optionendialog eingestellt haben.

Symbolleisten, Kürzel, Menüs beinhält Ihre Präferenzen, die Sie auf ihrer Seite im Optionendialog eingestellt haben.

Programmstatus enthält Dinge wie Fensterpositionen und zuletzt verwendete Dateien.

Sitzungen enthält einige oder alle Ihrer benannten Sitzungen.

Dateiformate enthält einige oder alle Ihrer eigenen, angepassten Dateiformate.

Profile beinhält einige oder alle Ihrer benannten Profile, die Einstellungen für FTP usw. enthalten.

4.Wenn Sie Sitzungen, Dateiformate oder Profile importieren, dann klicken Sie Weiter, um die jeweiligen einzubeziehenden Dinge auszuwählen.

Warnung  Ein Objekt wird rot dargestellt, wenn sein Name derselbe ist wie eines in Ihren gespeicherten Einstellungen.  Wenn Sie ein rotes Objekt importieren, wird es Ihre existierende Sitzung, Dateiformat oder Profil ersetzen!

Markieren Sie das Kontrollkästchen Alle bestehenden Sitzungen löschen, um alle Ihre gespeicherten Sitzungen zu löschen bevor jene in der Importdatei importiert werden.  Markieren Sie ebenso, wenn gewünscht, die Kontrollkästchen Alle bestehenden Dateiformate löschen oder Alle bestehenden Profile löschen.

5.Drücken Sie Fertig um die Einstellungen zu importieren.
