Wo die Einstellungen auf Ihrem Rechner gespeichert werden

Beyond Compare speichert Ihre Sitzungen, selbsterstellte Dateiformate und sonstige Präferenzen als XML-Dateien in seinem Einstellungen-Ordner.  Eine schnelle Möglichkeit, diesen zu finden, ist Hilfe > Unterstützung zu wählen, und dann die Verknüpfung Einstellungen-Ordner untersuchen am unteren Rand des Dialoges anzuklicken.

Standardinstallation

In Windows ist der Einstellungenordner innerhalb des Anwendungsdatenordners.  Der Ort ist von Ihrer Windows-Version abhängig.  Beispielsweise ist der Einstellungenordner unter Windows Vista hier:

C:\Users\<Benutzername>\AppData\Roaming\Scooter Software\Beyond Compare 4

 

Beachten Sie, dass sich dieser Ablageort zwischen verschiedenen Anwendern auf demselben Rechner unterscheidet.

Bei Mac ist der Einstellungenordner:

~/Library/Application Support/Beyond Compare

 

Sollten Sie mehrere Hauptversionen installieren wollen, dann können Sie diesen Ordner folgendermaßen umbenennen:

~/Library/Application Support/Beyond Compare 4

 

Bei Linux ist der Einstellungenordner:

~/$XDG_CONFIG_HOME/bcompare

 

Sollte $XDG_CONFIG_HOME nicht definiert sein, dann ist der Einstellungenordner:

~/.config/bcompare

 

Sollte eine ältere Version von Beyond Compare installiert sein, dann ist der Einstellungenordner:

~/.beyondcompare

 

Sollten Sie mehrere Hauptversionen installieren wollen, dann können Sie die Hauptversion an den Einstellungenordnernamen wie folgt anhängen:

~/$XDG_CONFIG_HOME/bcompare4

 

Portable Installation

Sollten Sie eine portablere Installation bevorzugen, mit Programm und Einstellungen an derselben Stelle, dann können Sie Ihre Einstellungsdateien in denselben Ordner wie das Programm kopieren.  Portable Installationen ermöglichen Ihnen, die Einstellungen auf einem Rechner für alle Anwender nutzbar zu machen, oder Ihre Einstellungen mitzunehmen.  Sollten Sie keine Einstellungsdateien zum Kopieren haben, können Sie eine leere Datei namens BCState.xml in dem Ordner erstellen.
