Einstellungenreferenz

Beyond Compare ist hochgradig konfigurierbar, und es ist manchmal nicht leicht, genau die Stelle im Programm zu kennen, an der eine bestimmte Einstellung geändert werden kann.  Hier ist aufgeführt, wo Sie nachsehen können:

Programmoptionen

Allgemeine Anwenderpräferenzen werden im Optionen-Dialog eingestellt.  Wählen Sie zum Öffnen Extras > Optionen .  Dort können Sie Startoptionen einstellen, Bestätigungsmeldungen steuern sowie Anzeigefarben und Schriftarten anpassen.

Sitzungseinstellungen

Die meisten der Optionen, die die Gültigkeit und Regeln eines Vergleichs definieren, befinden sich im Dialog für Sitzungseinstellungen.  Jeder Sitzungstyp besitzt seinen eigenen Dialog für Sitzungseinstellungen.  Wählen Sie Sitzung > Sitzungseinstellungen um den Dialog zu öffnen, oder klicken Sie einfach auf die Schaltfläche Regeln in der Symbolleiste.

Sitzungseinstellungen werden zusammen mit Ihren gespeicherten Sitzungen abgelegt.

Dateiformate

Bestimmte Dateitypen können im Dialog Dateiformate konfiguriert werden.  Wählen Sie zum Öffnen Extras > Dateiformate .

Profile

Bestimmte entfernte Dienste können im Dialog Profile konfiguriert werden.  Wählen Sie zum Öffnen Extras > Profile.

Integration von Versionskontrolle

Bestimmte Zuordnungen hinsichtlich Versionskontrolle können im Dialog für Integration von Versionskontrolle konfiguriert werden.  Wählen Sie zum Öffnen Extras > Integration von Versionskontrolle.
