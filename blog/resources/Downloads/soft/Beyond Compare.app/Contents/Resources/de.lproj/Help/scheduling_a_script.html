Zeitsteuerung eines Skripts

Sie können Beyond Compares Skript-Verarbeitung zur Ausführung bestimmter Tätigkeiten ohne Benutzerinteraktion verwenden, wie Synchronisation von Ordnern.  Um beispielsweise eine Sicherung des Ordners "C:\Mein Ordner" zu automatisieren, geben Sie die folgenden Skript-Befehle in einen Texteditor ein:

load "C:\Mein Ordner" "C:\Meine Sicherungen"

expand all

select left.newer.files left.orphan.files

copy left->right

 

Speichern Sie diese Datei unter "C:\Meine Skripte\Sync.txt".

Um ein Beyond Compare Skript zu einer gewünschten Zeit ablaufen zu lassen:

1.Wählen Sie in der Windows Aufgabenplanung Aktion > Einfache Aufgabe erstellen um den Aufgabenassistenten zu erhalten.
2.In Aktion wählen Sie Programm starten.
3.In Programm/Skript wählen Sie die Beyond Compare 4 Anwendung.
4.Im Eingabefeld Argumente hinzufügen fügen Sie eine Referenz zu Ihrer Skriptdatei hinzu:
@Synchronisation.txt
5.In Starten in wählen Sie den Skriptordner:
C:\Meine Skripte

Wir empfehlen Ihnen einen Log-Befehl in Ihr Skript einzubauen.  Sollten Sie mit Ihrem Skript Schwierigkeiten haben, überprüfen Sie das Fehler-Log für geplante Tasks in Ihrem Windows-Ordner: "SchedLgU.txt".
