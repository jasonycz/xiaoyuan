Sitzungen verwalten

Beyond Compare speichert Ihre letzten Sitzungen automatisch, so dass Sie sehr leicht auf sie zurück kommen können.  Möchten Sie eine Sitzung auf unbestimmte Zeit beibehalten, so können Sie sie unter einem beschreibenden Namen abspeichern.  Sie können Ihre benannten Sitzungen zudem in einer Ordnerstruktur organisieren.

Verwenden Sie die Startansicht, um alle Ihre Sitzungen von einer Stelle aus zu verwalten.  In der Baumstruktur für Sitzungen können Sie auf einer gespeicherten Sitzung einen Rechtsklick ausführen und Bearbeiten wählen, um den Verwaltungsbereich einzublenden.  Klicken Sie auf die Schaltfläche Sitzungsverwaltungsbereich anheften oder markieren Sie Ansicht > Sitzungsverwaltung anzeigen, um sie stets zwingend in der Startansicht anzeigen zu lassen.  Löschen und Umbenennen stehen auch im Rechtsklick-Menü für gespeicherte Sitzungen zur Verfügung.

Eine benannte Sitzung kann gesperrt werden, um sie gegen unbeabsichtigte Modifikationen zu schützen.  Führen Sie darauf einen Rechtsklick in der Baumstruktur für Sitzungen aus und wählen Sie Sperren.

Sobald eine Sitzung in einer Ansicht geöffnet wurde, enthält das Menü Sitzung allgemeine Befehle zu ihrer Handhabung.  Wenn Sie Änderungen an den primären Parametern einer benannten Sitzung vornehmen, werden Sie beim Verlassen des Programms darauf hingewiesen, und es besteht die Möglichkeit zum Speichern.

Um Standardsitzungseinstellungen zu ändern

Wählen Sie in der Startansicht in der Baumstruktur für Sitzungen unter dem Zweig Neu einen Sitzungstyp aus.  Führen Sie darauf einen Rechtsklick aus und wählen Sie Bearbeiten, um den Verwaltungsbereich einzublenden.  Änderungen dieser Standardeinstellungen werden ab jetzt für jede neu erstellte Sitzung dieses Typs verwendet.
