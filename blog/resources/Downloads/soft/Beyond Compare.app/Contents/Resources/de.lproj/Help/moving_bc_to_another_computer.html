Beyond Compare auf einen anderen Rechner verlegen

Um Beyond Compare auf einen anderen Rechner zu kopieren oder zu schieben

1.Laden Sie die letzte Version von Beyond Compare auf den neuen Rechner herunter, und installieren sie diese Version.
2.Geben Sie Ihren Registrierschlüssel in die neu installierte Version ein.
3.Starten Sie Beyond Compare auf dem alten Rechner und wählen Sie Extras > Einstellungen exportieren, um alle gewünschten Einstellungen in einer Datei zu speichern.
4.Kopieren Sie diese Einstellungendatei auf den neuen Rechner.
5.Starten Sie Beyond Compare auf dem neuen Rechner und wählen Sie Extras > Einstellungen importieren, um die Einstellungendatei zu laden.

Hinweis:  Wenn auf Ihrem alten Rechner Beyond Compare 2.0 oder älter installiert ist, steht Ihnen kein "Einstellungen exportieren"-Befehl zur Verfügung.  Zur Unterstützung hinsichtlich Übertragung Ihrer Einstellungen können Sie mit Scooter Software Kontakt aufnehmen.
