Startansicht - Übersicht

Sobald Sie ein neues Fenster oder einen neuen Tab öffnen, zeigt Beyond Compare die Startansicht an:

Von der Startansicht aus können Sie folgendes tun:

•Eine neue Vergleichssitzung beliebigen Typs starten
•Eine Sitzung aus den zuvor gespeicherten Sitzungen öffnen
•Eine automatisch gespeicherte Sitzung öffnen
•Nach einer Sitzung suchen
•Die Einstellungen für Ihre Sitzungen verwalten
•Die Standardeinstellungen für neue Sitzungen bearbeiten

Sie können Dateien und Ordner zum Vergleichen auch per "Drag-and-drop" von einem Dateimanager wie Windows Explorer auf diese Ansicht ziehen.  Ein "Drag-and-drop" auf eine der großen Schaltflächen wird eine Sitzung der jeweiligen Art starten.
