Understanding the Folder Merge Display

The Folder Merge display uses color cues to highlight the differences between the base folders.  By default, the color meanings are:

 

Gray

Unknown or older

 

Black

Same

 

Teal

Left change

 

Magenta

Right change

 

Orange

Mergeable

 

Red

Conflict

Pick View > Legend to display a guide to the various folder icon color combinations.

To change the default colors, pick Tools > Options , switch to the Colors, Fonts > Folder Views page, and find the Merge colors settings.

You can temporarily hide the Center Pane to give the left and right panes more room.
