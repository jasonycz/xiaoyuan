Folder Merge Overview

A Folder Merge session displays two or three folder structures side-by-side.  The left and right panes hold two versions of a folder structure.  Beyond Compare helps you combine changes from both versions into a specified output folder.  The optional center pane holds an older version of the folder structure, a common ancestor of the left and right versions.
